Luminara

A lightweight, plugin-based HTTP client for modern web applications built on native fetch

💡 Philosophy

Simple by Design: Like light traveling through space, Luminara embraces simplicity. A powerful HTTP client with native fetch at its core and optional driver support.

Separation of Concerns: Drivers handle communication, extensions live in core. Clean architecture, clear boundaries, harmonious design.

Developer-Friendly: Descriptive names, clear APIs, self-documenting code. Built with mindfulness for those who craft with intention.

Interactive Sandbox: Explore 21 examples across 8 features. Run tests individually or together. Live retry logging, backoff strategies, plugins, and custom drivers—all at your fingertips.

Illuminating the Path: Just as stars guide travelers, Luminara guides your HTTP requests with grace, reliability, and cosmic precision. ✨
